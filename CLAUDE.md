# Chimera

This is a configuration-only project (no code to build/test). It provides Claude Code rules and documentation for hybrid Claude + Codex orchestration.

## Structure

- `.claude/rules/chimera.md` — Core rules that instruct Claude to delegate to Codex CLI
- `examples/` — Usage examples and workflows
- `README.md` — Public documentation
- `CONTRIBUTING.md` — Contribution guidelines

## Conventions

- Keep rules concise and actionable
- Examples should be copy-pasteable
- Model routing table must stay current with available Codex models
