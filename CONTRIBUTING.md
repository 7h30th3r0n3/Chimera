# Contributing to Chimera

Thanks for your interest in Chimera! Here's how to contribute.

## Ideas & Discussions

- Open an **Issue** to suggest new use cases, model routing strategies, or integration patterns
- Share your Chimera configurations and workflows

## Pull Requests

1. Fork the repo
2. Create a feature branch (`git checkout -b feat/my-feature`)
3. Test your changes with both Claude Code and Codex CLI
4. Commit with conventional commits (`feat:`, `fix:`, `docs:`)
5. Open a PR with a clear description

## What we're looking for

- New example workflows in `examples/`
- Improved model routing strategies
- Support for additional AI CLI tools (Gemini CLI, Copilot CLI, etc.)
- Better delegation heuristics in `.claude/rules/chimera.md`
- Translations

## Code of Conduct

Be respectful. We're all here to build cool hybrid AI workflows.
